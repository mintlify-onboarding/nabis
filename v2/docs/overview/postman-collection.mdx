title: "Postman Collection"
---

We have provided a Postman collection that contains every parameter on every route. Make sure to set up a Postman environment with the variable `{{access-token}}`, which is set as a secret, with your token.

* [Download Collection](https://developers.nabis.com/v2/assets/NABIS-Platform-API-v2.postman%5Fcollection.json)
* [Postman Variables Quickstart](https://learning.postman.com/docs/sending-requests/variables/#variables-quick-start)
