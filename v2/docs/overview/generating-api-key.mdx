title: "Generating an API Key"
---

<Note>If you are interested in accessing this API for your organization, please reach out to [help@nabis.com](mailto:help@nabis.com).</Note>

To generate an API key to use:

1. Log in to the Nabis app.
2. In the left-hand navigation menu, select **Team**.
3. In the tabs across the page, select **API**.
4. Click the **Generate API Key** button.
5. Give your new API key a name.
6. Click the **Save** button.
